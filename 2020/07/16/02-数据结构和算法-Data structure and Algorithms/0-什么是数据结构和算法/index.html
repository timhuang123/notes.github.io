0-什么是数据结构和算法

阅读量: Tim Huang 2020-07-16 22:43:50

计算机内一个经典的公式，程序=数据结构+算法。而计算机专业做的就是写程序，所以数据结构和算法的重要性怎么强调都不过分。

数据结构 – Data Structure

数据结构：数据元素以一种特定的逻辑方式安排在一起。

常用数据结构

数组，链表，队列，栈，哈希表，集合，字典，树，堆，优先级队列，二叉搜索树，图。

算法 – Algorithm

算法：解决可计算问题的方法，好像没说一样。

常用算法

递归，分治，贪心，动态规划，回溯；DFS和BFS。
